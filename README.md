# Praktikum Teknik Kendali
Repository ini berisi kode program ESP32 maupun python yang digunakan dalam Mata Kuliah Praktikum Teknik Kendali, Program Studi Teknik Elektro, Universitas Maritim Raja Ali Haji

Praktikum terdiri dari 5 materi yaitu:

1. FILTERING SENSOR DAN AKUISISI DATA

2. ANALISIS RESPON SISTEM DAN KONTROL PROPORSIONAL (P)
   
3. IMPLEMENTASI DAN TUNING PID PADA ESP32

4. PERANCANGAN FUZZY CONTROL

5. IMPLEMENTASI FUZZY EMBEDDED PADA ESP32
